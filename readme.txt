    Проект store - представляет собой реализацию тестового задания.

    Тестовое задание - разработка витрины Интернет-магазина (каталог продуктов).

    Приложение состоит из следующих частей:
1. База данных на платформе PostgreSQL
2. Web-сервис, принимающий запросы от клиентских программ по протоколу http
3. Web-интерфейс для работы с витриной магазина и его администрирование

    Перед началом работы с приложением следует создать базу данных
СУБД Postgres, используя скрипт create-db.sql, расположенный в директории scr/main/resources/sql

    Также выполнить настройку соединения к БД в файле db-settings.properties,
расположенный в директории  scr/main/resources/settings

 (по умолчанию)
 наименование БД - store
 пользователь - postgres
 пароль - указанный при регистрации пользователя postgres.

    Для запуска приложения в адресной строке веб-браузера следует указать
 http://localhost:8080/store/

    Для тестирования rest-сервиса и dao были разработаны тесты в пакетах
    home.ivishnyakova.store.dao
    home.ivishnyakova.store.rest

    Для выполнения тестирования можно использовать плагин tomcat7-maven-plugin
для развертывания приложения из среды Idea:
    http://localhost:8888/store
